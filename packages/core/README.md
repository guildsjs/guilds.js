<h1 align="center">@guilds/core</h1>

guilds.js is a Node.js library for using Discord's API and gateway.

## Installation

```bash
npm install guilds@/core
# or
pnpm add guilds@/core
# or
yarn add @guilds/core
# or
bun add @guilds/core
```
